# factory.ts-sample

- Sample code for [factory.ts](https://www.npmjs.com/package/factory.ts). 

## System Requirements

- Node.js - 14.x
- Yarn - 1.22.x

## Library

- TypeScript - 4.5.x
- factory.ts - 0.5.x

## Usage

```bash

yarn

yarn build

yarn start
```
